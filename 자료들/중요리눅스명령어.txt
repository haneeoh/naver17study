0. clear : 화면 지우기
0. history : 사용했던 모든 명령어 보여주기
1. man ls  : ls 시용법 확인
2. pwd : 현재 위치 확인
3. ls : 목록보기
     -a  : 숨김파일이나 폴더까지 모두 출력
     -l : 목록형식으로 세로방향으로 출력
     -h: 크기를 바이트에서 M 단위로 변경해서 출력
     -S: 크기의 내림차순으로 출력
4. cd : 디렉토리 변경
     cd ~ :꼭대기(루트)로 이동
     cd /d/naver1210  : d드라이브의 naver1210폴더로 이동
     cd - :직전 디렉토리로 이동
5. mkdir : 디렉토리 생성
      -p  : 하위디렉토리까지 한번에 생성
     mkdir hello : hello 디렉토리 생성
     mkdir -p bitcamp/member/study  :하위디렉토리까지 한꺼번에 생성
6. rmdir  : 비어있는 디렉토리 삭제(비어있지 않을경우 오류)
7. rm: 파일이나 디렉토리 삭제
        -r : 디렉토리 삭제시 하위 디렉토리도 같이 삭제
        -f : 삭제물음없이 강제 삭제
8. cat : 파일안의 내용 확인
    -n :번호출력
9. cp :복사
    -r : 하위 디렉토리까지 모두 복사
    -b : 복사될 파일이름이 이미 존재할때 백업파일을 생성
    -f  : 복사될 파일이름이 이미 존재할때 강제로 덮어쓰기
    -i  : 복사될 파일이름이 이미 존재할때 덮어쓰기 여부를 묻는다
10. vi 에디터 : 문서 편집
    :w : 저장
    :q  or :q!  :종료
    :wq : 저장후종료
    i or a : 편집 모드
    esc 키 : 명령 모드
   dd : (명령모드)한줄삭제(잘라내기)
   p : (명령모드)붙여넣기
   :set nu : 라인번호(취소 :set nonu)
   /문자열 : (명령모드)해당 문자열로 이동
   :숫자 : 해당라인으로 이동
   줄의 맨앞 : 0(숫자 0)
   줄의맨뒤 :$
